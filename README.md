         Инструкция
- Установить Python 3.9
- Запустить файл install_libs.cmd
- Распределить искодники по папкам в папке src
- Указать в порядке возростания (1 - фон, 2 - след., 3...) названия папок в файле config.json
- Запустить start.cmd
